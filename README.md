# Hacker News

Установить Node.js:
`https://nodejs.org/en/download/prebuilt-installer`
Установить yarn: `npm install --global yarn`

1. Скачать проект:

   `git clone 'https://github.com/serg310583/hacker-news.git`

2. Перейти в папку с проектом:

   `cd news`

3. Установить зависимости:

   `yarn install`

4. Запустить проект:

   `yarn dev `
